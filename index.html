NyikatBuos Shoe cleaning service Website

NyikatBuos is a website project that I designed independently for a small business (UMKM) specializing in shoe cleaning services. This website provides essential information about the business, including its background, available services, and pricing details, making it easier for customers to understand and access their offerings.

The website features a clean and user-friendly interface with a responsive design, ensuring a seamless experience across different devices. Customers can browse the pricing list for various shoe cleaning services and learn more about the company’s commitment to quality service. The goal of this project is to enhance NyikatBuos' online presence, making their services more accessible to a wider audience.

Alta Tech Mobile app Development

One of the main challenges faced by retail brands implementing a membership system is the lack of attractive benefits offered to customers. Additionally, customer loyalty towards a brand tends to be low, partly due to the complicated membership registration process and the lack of transparency in discount features. To address these issues, we developed AltaTech Store, a mobile application designed to enhance the shopping experience by providing clear membership benefits, improving customer engagement, and simplifying the registration process.

As a Front-End Mobile Developer, my role in this project is to ensure a seamless and user-friendly interface, making it easier for users to access membership features and enjoy the benefits offered by the brand.

Tobacco Trade Information System

The Tobacco Merchant Information System is a web-based project I developed as my final university project to help merchants manage their inventory, sales, and transactions more efficiently. This system replaces the traditional method of using Excel spreadsheets with a user-friendly digital platform, improving data accuracy and accessibility.

The website features a responsive and intuitive interface, built using HTML, CSS, JavaScript, and Bootstrap, ensuring seamless access across different devices. The interactive dashboard provides merchants with real-time insights into sales, inventory levels, reports, and financial summaries, helping them make informed business decisions By transitioning merchants to this integrated system, the project enhances operational efficiency and streamlines data management, making it easier for them to track and analyze business performance.
